# 超哥通讯
## 开发环境
部署环境：jdk1.8+  tomcat7+ mysql5.6+
开发工具：idea hbuilderx
服务端：springboot+mybatis+mysql+maven
客户端：使用Hbuilder的mui框架实现的H5+App 可直接连接手机调试，也可打成发布成apk或ipa安装包

##帮助：可百度MUI查看mui开发文档hbuilder中有mui演示项目，可新建一个从其中找到自己所需的控件进行使用

##运行步骤：
1. 导入数据库文件
2. 部署服务端并运行
3. 在hbuilder上打开客户端，连接手机运行app（或直接将apk安装到手机）
4. 打开app，在开始页面点击齿轮设置服务端访问路径，保存并重新启动即可
5. 注册账户，注册成功登录。
